Описание

Пример данных

        {
            "avatar": "https://randomuser.me/api/portraits/men/93.jpg",
            "firstname":"Тимур",
            "lastname":"Терентьев",
            "additionalInfo":"Белгородский государственный национальный исследовательский университет",
            "domain":"https://vk.com/2domain_name"
        }

Пример использования с клиентскими данными

        initDropdown('#dropdown1', {
             data: [...],
             multiSelect: true,
             showAvatar: true
        });

Пример использования с серверными данными

        initDropdown('#dropdown2', {
             url: url,
             multiSelect: true,
             showAvatar: true
        });

Демо

Данные на клиенте с выбором нескольких пользователей и отображение аватара

Данные на сервере с выбором нескольких пользователей и отображение аватара

Данные на клиенте с выбором одного пользователя и отображение аватара

Данные на клиенте с выбором нескольких пользователей и без отображение аватара

Данные на сервере с выбором одного пользователя и без отображение аватара
